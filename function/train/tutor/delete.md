# 删除导师

## 入口

点击表格内“操作”列的“删除”时，会弹出如下确认对话框：

![删除导师](<../../../.gitbook/assets/image (6).png>)

## 规则

删除后，该导师的可用状态将改为“已删除”，并且：

1. 未来新增的导师可以使用这个导师名称。

{% hint style="info" %}
注意：已删除的导师无法恢复成正常状态。
{% endhint %}

{% hint style="info" %}
为了确保前端交互体验的完整性和一致性，删除导师后，仍可以在场馆端App的课程详情中看到导师的信息。
{% endhint %}
