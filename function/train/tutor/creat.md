# 新增导师

## 入口

在列表中点击“新增”按钮后，打开如下对话框：

![新增导师](<../../../.gitbook/assets/image (28).png>)

## 规则

| 数据 | 是否必填 | 规则                                                   |
| -- | ---- | ---------------------------------------------------- |
| 名称 | 必填   | <p>1~24个任意文字<br>不能与其他未删除的导师重名</p><p>名称显示在场馆端App中</p> |
| 图片 | 必填   | 1M以内的jpg或png图片                                       |
| 介绍 | -    | 1\~500个任意文字                                          |

{% hint style="info" %}
图片建议尺寸（或等比例尺寸）：

导师图标：120px × 120px
{% endhint %}
