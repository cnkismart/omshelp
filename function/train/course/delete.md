# 删除课程

## 入口

点击表格内“操作”列的“删除”时，会弹出如下确认对话框：

![删除课程](<../../../.gitbook/assets/image (6).png>)

![删除有关联轮播图或推荐位的课程](<../../../.gitbook/assets/image (17).png>)

## 规则

删除后，该课程的可用状态将改为“已删除”，并且：

1. 场馆端App中不再显示这个课程；
2. 未来新增的课程可以使用这个课程名称。
3. 如果有轮播图或推荐位关联了该课程，在删除后也将同时删除这些轮播图和推荐位。

{% hint style="info" %}
注意：已删除的课程无法恢复成正常状态。
{% endhint %}
