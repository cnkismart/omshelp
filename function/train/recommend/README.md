# 推荐位管理

## 功能说明

该模块是用来管理练多多2.0场馆端App“在线微课”板块的推荐位。

目前有4个固定的推荐位，当前版本不支持新增、停用和删除。但可以清空推荐位关联的内容，在场馆端App中只会显示关联内容的推荐位。

{% hint style="info" %}
若中间有未关联内容的推荐位，后面的推荐位会往前补齐位置。

比如：如果只有“1”、“4”两个推荐位有关联内容，在场馆端App中展示时，“4”推荐位的内容会显示在“2”推荐位的位置。
{% endhint %}
