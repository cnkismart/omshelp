# 停用产品分类

## 入口

点击表格内“操作”列的“停用”后，弹出如下确认对话框：

![停用确认对话框](<../../../.gitbook/assets/image (13).png>)

{% hint style="info" %}
只有正常状态的产品分类，在“操作”列才会显示“停用”菜单。
{% endhint %}

## 规则

停用成功后，该产品分类将被改为“已停用”状态，并且：

1. 新增产品时不能再选择这个产品分类。

{% hint style="info" %}
已停用的产品分类仍可以重新启用，详见：[启用分类](enable.md)。
{% endhint %}
