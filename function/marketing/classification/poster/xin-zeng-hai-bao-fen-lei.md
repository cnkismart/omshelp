# 新增海报分类

## 入口

在列表中点击“新增”按钮后，打开如下对话框：

![新增海报分类对话框](<../../../../.gitbook/assets/image (42).png>)

## 规则

| 数据   | 是否必填 | 规则                                                    |
| ---- | ---- | ----------------------------------------------------- |
| 展示名称 | 必填   | <p>1~6个任意文字<br>不能与其他未删除的海报分类重名</p><p>名称显示在场馆端App中</p> |
| 图片   | 必填   | 1M以内的jpg或png图片                                        |

{% hint style="info" %}
图片建议尺寸（或等比例尺寸）：

分类图标：82px × 82px
{% endhint %}
