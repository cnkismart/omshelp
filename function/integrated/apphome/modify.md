# 修改首页宣传图

## 入口

在列表表格的“操作”列，点击“修改”后，打开如下对话框：

![修改对话框](<../../../.gitbook/assets/image (1).png>)

## 规则

| 数据 | 是否必填 | 规则                                            |
| -- | ---- | --------------------------------------------- |
| 名称 | 必填   | <p>3~8个任意文字，不能与其他位置的名称重复</p><p>不显示在场馆端App</p> |
| 图片 | 必填   | 1M以内的jpg或png图片                                |

{% hint style="info" %}
图片建议尺寸（或等比例尺寸）：

在线微课：666px × 142px

精选活动：250px × 310px

创意海报：402px × 142px

场馆险：194px × 154px

器械资讯：194px × 154px

行业口碑：666px × 142px
{% endhint %}
